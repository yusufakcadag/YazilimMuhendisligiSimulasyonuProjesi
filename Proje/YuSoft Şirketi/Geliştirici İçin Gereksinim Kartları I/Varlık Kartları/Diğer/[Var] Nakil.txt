----------------------------------------------------------------------------------------------------
[Var] Nakil
----------------------------------------------------------------------------------------------------
Özellikler:
	- ID 0*
	- Aktiflik 1*
	- Taşıt ID
	- Şoför ID
	- Nakil Başlangıç Tarihi
	- Nakil Bitiş Tarihi
	- Olaylar
	- Ücret
----------------------------------------------------------------------------------------------------
0*: ID şu şekilde oluşturulur: Taşıtın ID'sine Nakil Başlangıç Tarihi eklenir.
Örneğin: ID 34ABC202206282022 ise Taşıtın ID'si 34ABC2022, nakil başlangıç tarihi 06282022'dir.

1* Nakil tamamlanmamışsa 1'dir tamamlanmışsa 0'dır.
----------------------------------------------------------------------------------------------------
İlişkili Varlıklar:
	- Olay
